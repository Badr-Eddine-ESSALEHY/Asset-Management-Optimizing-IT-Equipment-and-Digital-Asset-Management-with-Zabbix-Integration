Quick Help

  • Connection Failed: Check Zabbix server is running and API is accessible
  • SNMP Test Failed: Verify IP address and SNMP is enabled on target device
  • No Monitoring Data: Wait a few minutes for data collection to start
  • Permission Errors: Ensure Zabbix user has proper permissions

  1. Ensure equipment has valid IP address configured
  2. Enable SNMP on the target equipment
  3. Enable monitoring checkbox in equipment settings
  4. Use "Setup Monitoring" button to sync with Zabbix
  5. Wait for data collection to begin (2-5 minutes)
